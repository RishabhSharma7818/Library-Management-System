# Library Management System (Java)

This is a simple Library Management System built using Java and OOP concepts.  
It includes classes for Books, Users, and Library operations such as issuing and returning books.

## Features
- View all books
- Issue a book
- Return a book
- View users (dummy data)

## Technologies Used
- Java
- Eclipse IDE

## How to Run
1. Download the source code from the repository.
2. Extract the ZIP file.
3. Open the folders in Eclipse.
4. Run `LibraryManagementSystem.java` as a Java Application.
